## Markdown on GitHub

- Just like this page, it is written using the markdown syntax, a way to style text on a page using the ***.md*** or ***markdown extension***.

- For headers or titles:
   > # \#Large\# # 
   > ## \##Medium\## ##
   > ### \###Small\### ###
   
- For emphasis of texts:
   > *\*italics\**
   
   > _\_italics\__
